## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Then, run prism a next-generation Node.js and TypeScript ORM:

```bash
npx prisma init
```

```bash
npx prisma generate
```

```bash
npx prisma db push
```

- Explore the data in Prisma Studio

```bash
npx prisma studio
```
